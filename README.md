## 📌 프로젝트 설명

본 프로젝트는 **Streamlit**을 활용하여 제작된 **대화형 데이터 분석 도구**입니다. 사용자가 CSV 파일을 웹에 업로드하면, 복잡한 코드 작성 없이도 데이터의 기본 정보를 확인하고, 컬럼별 분포를 시각화하며, 원하는 기준으로 데이터를 그룹화하여 집계할 수 있습니다.

특히 한국어 환경에서 자주 발생하는 **인코딩 문제(CP949, EUC-KR)를 자동으로 처리**하고, 사용자가 직접 **헤더(Header) 위치를 지정**할 수 있도록 하여 데이터 로드 편의성을 높였습니다.

## 📊 주요 기능 (데이터 분석 내용)

본 애플리케이션은 사용자가 업로드한 CSV 파일을 기반으로 다음과 같은 분석 기능을 제공합니다.

### 1. 스마트한 파일 로드
* **CSV 파일 업로드**: 사이드바를 통해 로컬의 CSV 파일을 업로드합니다.
* **인코딩 자동 감지**: `UTF-8`, `CP949`, `EUC-KR` 등 자주 사용되는 인코딩을 자동으로 감지하여 파일을 읽어 들입니다. 실패 시 수동 입력도 지원합니다.
* **헤더 위치 지정**: 데이터가 시작되는 행(헤더)의 번호를 사용자가 직접 지정할 수 있습니다.

### 2. 데이터 기본 정보 (Data Overview)
* **데이터 원본 보기**: `df.head(N)` - 사용자가 원하는 상위 N개의 행을 직접 확인할 수 있습니다.
* **데이터 Shape**: `df.shape` - 전체 데이터의 행과 열 개수를 표시합니다.
* **데이터 기본 정보**: `df.info()` - 각 컬럼의 데이터 타입, Non-null 개수 등을 요약합니다.
* **기초 통계량**: `df.describe()` - 수치형 데이터의 평균, 표준편차, 사분위수 등 기초 통계량을 보여줍니다.

### 3. 컬럼 상세 분석 (Single Column Analysis)
* **컬럼 자동 감지**: 사용자가 선택한 컬럼이 **숫자형**인지 **범주형**인지 자동으로 판단합니다.
* **숫자형 (Numeric)**: **히스토그램(Histogram)**을 통해 데이터의 분포를 시각화합니다.
* **범주형 (Categorical)**: **막대그래프(Bar Chart)**를 통해 각 값의 빈도(Value Counts)를 시각화합니다.

### 4. 사용자 정의 그룹화 분석 (Groupby Analysis)
* 사용자가 직접 **그룹 기준 컬럼(GroupBy)**, **집계 대상 컬럼(Target)**, **집계 방식(Aggregation)**을 선택하여 데이터를 요약합니다.
* **그룹 기준**: 하나 이상의 컬럼을 선택하여 다중 그룹화(MultiIndex)가 가능합니다.
* **집계 방식**: `mean`, `sum`, `count`, `max`, `min` 등 다양한 집계 함수를 지원합니다.
* **결과**: 집계된 결과는 **데이터 테이블**과 **막대그래프**로 즉시 시각화됩니다.
